# Streaming Service Program (Java OOP)

A desktop-based streaming service system developed using Java.

## Features
- Manage Movies, Series, and Podcasts
- Add and remove media dynamically
- Episode management for series (Array & Composition)
- Subscription system with multiple plans
- Payment calculation including VAT
- File handling (read/write data)
- GUI using Java Swing

## Technologies Used
- Java
- OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
- ArrayList & Arrays
- Exception Handling
- File Handling
- Java Swing (GUI)

## Project Overview
This project demonstrates the implementation of a complete object-oriented system, including inheritance hierarchies, interfaces, abstract classes, and user interaction through both console and GUI.
